Modul ini bertujuan agar mahasiswa dapat menginstal dan menggunakan Wireshark untuk menangkap serta menganalisis paket data jaringan. Pemahaman protokol jaringan tidak cukup hanya teori, tetapi perlu dilihat secara langsung melalui proses komunikasi antar entitas di jaringan. Oleh karena itu, Wireshark digunakan sebagai alat untuk mengamati bagaimana protokol bekerja secara nyata di internet.

Langkah menggunakan wireshark:
1.Pertama buka wireshark
![langkah_1](../assets/week%202/1.png)

2.Buka file pdf modul jarkom lalu salin lik dari modul 2 http://gaia.cs.umass.edu/wireshark-labs/INTRO-wireshark-file1.html.
![langkah_2](../assets/week%202/2.png)

3.Kemudian kembali ke Wireshark dan ketik **http** pada kolom pencarian (tanpa tanda kutip). Setelah itu, pilih paket yang memiliki keterangan **(text/html)** (tanpa tanda kutip).
![langkah_3](../assets/week%202/3.png)
